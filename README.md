# Appointment_Scheduler

A desktop-based Java application designed for efficient management of appointments and shared resources such as meeting rooms or medical consultations. This system supports multi-user roles, ensures conflict-free scheduling, and utilizes a MySQL database for persistent storage.

## Key Features

- **User Authentication**: Secure registration and login with role-based access control (Admin and User).
- **Appointment Booking**: Intuitive interface for selecting dates, times, and specific resources.
- **Conflict Management**: Automated detection to prevent overlapping bookings.
- **Administrative Control**: Dedicated panels for managing users, viewing all appointments, and maintaining resource lists.
- **Modular Architecture**: Clear separation between the graphical user interface (Swing) and the underlying business logic.

## Technical Stack

- **Language**: Java
- **User Interface**: Java Swing
- **Database**: MySQL
- **Connectivity**: JDBC (Java Database Connectivity)

## Project Structure

The application demonstrates core software engineering principles:
- Object-Oriented Design (OOD)
- JDBC-based database interactions
- Graphical User Interface development in Java
